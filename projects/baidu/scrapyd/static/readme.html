爬虫说明

 

百度网盘组织结构

百度网盘的包含用户，我们称之为user。每个用户有一些对外开放的分享数据，一条分享数据可能是一个文件也可能是一个目录，所以这里一条分享是一个抽象的概念。我们称之为share。每条分享数据有他的数据名称，有他包含的文件/文件夹。这些分享的文件/文件夹我们称之为file。

 每个user有一些粉丝，也有他关注的其它用户，爬虫就是利用抓取已知每个用户粉丝和关注列表来发现新用户。

搜索生成url说明

如果shorturl 不为空：http://pan.baidu.com/s/+${shorturl }

例如

http://pan.baidu.com/s/1eQnGnuU

如果shorturl 为空：

如果feed_type=share:

http://pan.baidu.com/share/link?shareid=${shareid}&uk=${uk}

例如

http://pan.baidu.com/share/link?shareid=1807079184&uk=1261733552

如果feed_type=album:

http://pan.baidu.com/pcloud/album/info?uk=${uk}&album_id=${album_id}

例如

http://pan.baidu.com/pcloud/album/info?uk=54898840&album_id=428507060389421354

 

注意：第二种方法对于所有都有效
